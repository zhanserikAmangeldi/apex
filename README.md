# Apex

Apex это современная платформа для самостоятельного обучения по методике zettelkasten с интеграций искусственного интеллекта. Наша главная задача помочь с структурированием знаний человека и ускорить его обучение за счет предоставления обратной связи, в виде статистики, рекомендаций для последующих тем для изучения, карт для повторения(по типу anki) и т.д.

## Функционал предоставляемый пользователю:
- Человек создает топик(т.е. хранилище)
- Может там создавать Markdown файлы, заливать туда файлы разных типов(pdf, mp4 и т.д.). Структура будет как у файлового дерева.
- Может создать свой dashboard и настраивать там виджеты, они и будут представлять нашу обратную связь. Для примера, виджет с статистикой, какую именно он хочет получать и т.д.
- Если он добавил интерактивный виджет, к примеру “вопрос дня”, то он сможет, ответить на вопрос, а мы это сохраним и добавим в контекст обучения.
- Так же он сможет посмотреть дерево связи своих файлов(гиперссылки или же тэги)
- Может найти информацию по своим заметкам за счет контекстного поиска

> Кроме этого есть идея, добавить возможность коллаборационной работы над топиком.

## Техническая часть:
- Frontend: React
- Backend: Golang + Python
- База данных: Postgresql
- будем обновлять по добавлению
